# Foundation Micro-Site Project.

### Use this Project as a base to create new Boost Micro-Sites.

Framework: [Next.js](https://nextjs.org/) \
API: BrandsAPI (AWS Boost System)

## Getting Started

~ To start a new Boost Micro-Site follow the next steps.

### CLONE REPOSITORY:

```bash
$ git clone https://github.com/Retail-Zoo/boost-microsite-base.git
```

### Dependencies:

```bash
# in console run

# DEPENDENCIES:
$ yarn install

# SERVER:
$ yarn dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Pages

Into this folder create all your new next-js pages.

> Example:

```tsx
// pages/index.txs

// NODE_MODULES
import type { NextPage } from 'next';
import Image from 'next/image';
// VENDOR
import Layout from '../vendor/components/layout/Layout';
import View from '../vendor/components/layout/View';
import Animation from '../vendor/components/ui/animation/Animation';
import FixedContainer from '../vendor/components/ui/FixedContainer';
import { getImagePath } from '../vendor/utils/files-herper';

const Header = () => (
  <View horizontal className="vh-10 text-bg-primary">
    HEADER
  </View>
);
const Footer = () => (
  <View horizontal className="vh-10 text-bg-secondary">
    FOOTER
  </View>
);
const Background = () => (
  <View fill>
    <Animation folder="fallen-fruits">
      <FixedContainer width={350} height={350}>
        <Image
          src={getImagePath('boostLogo.svg')}
          width={155}
          height={57}
          alt="Boost Logo"
        />
      </FixedContainer>
    </Animation>
  </View>
);

const HomePage: NextPage = () => (
  <Layout
    title="Home Page"
    backgroundColor="#282828"
    textColor="cyan"
    header={Header}
    footer={Footer}
    background={Background}>
    <View fill left bottom className="container vh-80">
      <div className="card mb-3">
        <div className="card-body">
          <h1 className="display-1 text-dark">HOME PAGE</h1>
        </div>
      </div>
    </View>
  </Layout>
);

export default HomePage;

```

## Components Folder

Create here all React Js components. <br>
Use [parseClassesName(args:array)](./vendor/utils/code-helper) to include [bootstrap](https://getbootstrap.com/) classes

> Example

```tsx
// src/components/MyButton.tsx

import React, { FC } from 'react';
import { bootstrapClass } from '../../vendor/utils/code-helper';

// bootstrap classes
const classNames = {
  button: bootstrapClass(['btn', 'btn-primary', 'px-3']),
  text: bootstrapClass(['fw-bolder', 'text-white']),
  icon: bootstrapClass([
    // fontawesome
    'fa-solid',
  ]),
};

// javascript element.style.[parameters]
const styles = {
  button: {
    textShadow: '1pt 1pt 1pt rgba(40,40,40,0.33)',
  },
};

type MyButtonType = {
  text?: string;
  icon?: string;
  onClick?: () => void;
};

// eslint-disable-next-line react/function-component-definition
const MyButton: FC<MyButtonType> = ({ text, icon, onClick }) => (
  <button onClick={onClick} className={classNames.button} style={styles.button}>
    {text && <span className={classNames.text}>{text}</span>}
    {icon && <i className={`${classNames.icon} fa-${icon}`} />}
  </button>
);

MyButton.defaultProps = {
  text: undefined,
  icon: undefined,
  onClick: undefined,
};

export default MyButton;
```

## GraphQL Folder

Create here your GraphQL queries and mutations. <br>
Then use the script to create all hooks to communicate with Boost BrandsAPI.

> Example

```tsx
// src/graphql/mutations.ts

import { gql } from '@apollo/client';

export const SUBMIT_ENTRY_MUTATION = gql`
  mutation SubmitEntry($input: SubmitEntryInput!) {
    submitEntry(input: $input) {
      entry {
        id
        acceptTermsAndConditions
        prize {
          id
          title
          description
          imageUrl
          digital
          couponHash
        }
      }
    }
  }
`;

export const UPDATE_ENTRY_MUTATION = gql`
  mutation UpdateEntry($input: UpdateEntryInput!) {
    updateEntry(input: $input) {
      entry {
        id
        prize {
          id
          title
          description
          imageUrl
          digital
          couponHash
        }
      }
    }
  }
`;
```

> Automatic GraphQL Code-Generation

```bash
# Ensure BrandsAPI is running locally
# http://localhost:4000/graphql
$ yarn codegen
```
